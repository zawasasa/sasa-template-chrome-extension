# アイコンファイル

Chrome拡張機能「ささっとテンプレ」のアイコンファイルです。

## 必要なファイル

以下のサイズのPNGファイルが必要です：

- `icon-16.png` - 16x16px (ツールバー用)
- `icon-32.png` - 32x32px (Windows用)
- `icon-48.png` - 48x48px (拡張機能管理ページ用)
- `icon-128.png` - 128x128px (Chrome Web Store用)

## アイコン作成手順

1. 提供された円形の紫色アイコン画像を保存
2. 各サイズに変換してPNGファイルとして保存
3. ファイル名を上記の通りに設定

## 使用アイコン

文書とテンプレートを表現した紫色の円形アイコン：
- 白い文書アイコン
- 緑色のテンプレートカード
- 矢印でテキスト挿入を表現
- 紫色のグラデーション背景
