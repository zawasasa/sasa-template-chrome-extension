# ささっとテンプレ

Chrome拡張機能「ささっとテンプレ」は、Webサイト上のテキスト入力欄で定型文を素早く挿入できる支援ツールです。

## 📋 主な機能

- **カテゴリー管理**: テンプレートを「メール用」「挨拶」などのカテゴリーで整理
- **テンプレート管理**: 定型文の作成、編集、削除
- **ワンクリック挿入**: テンプレートボタンをクリックするだけで入力欄に挿入
- **ローカル保存**: データはPC内のChromeにのみ保存（同期機能なし）

## 🚀 インストール方法

1. Chrome で `chrome://extensions/` を開く
2. 右上の「デベロッパーモード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. このフォルダを選択

## 💡 使い方

1. **拡張機能を開く**: ブラウザのツールバーの「ささっとテンプレ」アイコンをクリック
2. **カーソルを合わせる**: 使用したいWebページの入力欄にカーソルを合わせる
3. **カテゴリーを開く**: サイドバーでカテゴリーの「▼」をクリック
4. **テンプレート挿入**: 使いたいテンプレートボタンをクリック

## 🛠️ 管理機能

### カテゴリー管理
- **追加**: 「新しいカテゴリーを追加」ボタンから作成
- **編集**: カテゴリーヘッダーの編集アイコンをクリック
- **削除**: カテゴリーヘッダーの削除アイコンをクリック

### テンプレート管理
- **追加**: 各カテゴリー内の「新しいテンプレートを追加」ボタンから作成
- **編集**: テンプレート右側の編集アイコンをクリック
- **削除**: テンプレート右側の削除アイコンをクリック

## 📝 制限事項

- カテゴリー名：20文字以内
- テンプレート名：20文字以内
- テンプレート内容：文字数制限なし
- 検索機能：なし
- バックアップ機能：なし

## 🎨 特徴

- **カード形式UI**: カテゴリーごとに色分けされた美しいカードデザイン
- **直感的操作**: テンプレートがボタン形式で分かりやすい
- **スムーズアニメーション**: ホバー効果とアニメーションで快適な操作感

## 🔧 技術仕様

- **Manifest Version**: 3
- **権限**: storage, activeTab
- **対応サイト**: 全てのWebサイト
- **データ保存**: Chrome Storage API (ローカル)

## 🐛 既知の問題

- 一部のWebサイトでテキスト挿入が動作しない場合があります
- 拡張機能の初回使用時に「接続エラー」が表示される場合は、拡張機能を再読み込みしてください
